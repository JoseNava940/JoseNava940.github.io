Bienvenido a API Inventarios

En esta API podrás visualizar, editar, agregar y eliminar categorias y productos de supermercado con el que podrás tener un mejor control de los mismos.

Lleva el control de las categorias y productos de un supermercado de forma rápida y sencilla actualizandose en tiempo real.

Administra las categorias de tu supermercado, por ejemplo, frutas y verduras, carnes, tecnología, electrónica, etc. y manten ordenado tu inventario.

Asigna una categoria a tus productos para que los puedas desplegar con base en esa categoría y lleva control preciso de toda la tienda.
